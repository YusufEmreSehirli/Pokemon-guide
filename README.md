# Pokemon-guide

<h1> Pokemon Cards </h1>

Bu uygulama, HTML, CSS ve JavaScript kullanılarak oluşturulmuş basit bir Pokémon listesi içerir. Ayrıca, PokéAPI kullanılarak gerçek zamanlı veri çekme işlevselliği sağlar.



 <h3> Teknolojiler </h3>

- HTML5
- CSS3
- JavaScript
- PokéAPI (https://pokeapi.co/)
